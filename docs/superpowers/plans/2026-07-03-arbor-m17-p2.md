## Task 1: Typed-ancestor staleness propagation + `move`/`remove` index hooks

**Files:** modify `src/semantic-index.ts`, `src/mutator.ts`; create `test/m17-ancestor-staleness.test.ts`

**Semantics locked:** a node under a typed-`embedText` ancestor is a suppressed shard — the ANCESTOR is the semantic unit and its embed text derives from its whole subtree. Therefore ANY subtree change (shard set/insert/remove, or a node moving in/out) must re-hash the ancestor. Mechanism: (a) `SemanticIndex.onChange`'s suppressed-shard branch also recurses into the owning ancestor; (b) `Mutator.move` fires `onChange` for the moved node + its descendants + BOTH parents; (c) `Mutator.remove` fires `onChange` for the parent. Firing `onChange` on an untyped container is a safe no-op-ish path (its embed text is null → state "none") and — when that container sits under an embedText unit — the suppressed-shard branch propagates up, which is exactly the desired transitivity.

Cases — each: mutate → `expect(index.staleCount())…` → `await reindex()` → search reflects the change:
  1. **Shard set:** insert a typed node (e.g. `{ title: "old", body: "b" }` type "doc" whose embedText concatenates fields) under `/docs`; reindex; then `set` ONLY `/docs/a/title` to `"new"`. EXPECT: the typed node is re-marked stale (pre-fix: stays fresh with the old vector) and after reindex a search for the NEW title text ranks it.

  4. **Move OUT:** a shard moved OUT of the typed subtree to the root. EXPECT: the old ancestor is stale; the moved node becomes independently embeddable (stale, then fresh+searchable after reindex).

- [ ] **Step 2 — `src/semantic-index.ts`.** Extract the ancestor walk and propagate:

```ts
  /** The nearest ancestor whose type declares embedText — the semantic unit that
   *  owns this shard — or undefined. */
  private embedTextAncestor(node: ArbNode): ArbNode | undefined {
    let pid = node.parentId;
    while (pid !== null) {
      const anc = this.tree.get(pid);
      const ancDef = anc?.type ? this.registry?.get(anc.type) : undefined;
      if (ancDef?.embedText) return anc;
      pid = anc?.parentId ?? null;
    }
    return undefined;
  }
```

In `onChange`, the suppressed-shard branch gains one propagation call before `return`:


(Recursion terminates: the ancestor has its own embedText, so it takes the normal branch. `isSuppressedShard` can now delegate its walk to `embedTextAncestor` — do so to avoid duplication.)

## Task 3: O(1) `byPath` child lookup + snapshot-restore single clone

**Files:** modify `src/artifact-tree.ts`, `src/addressing.ts`; create `test/m17-child-lookup.test.ts`

**Design locked (invalidate-on-write, rebuild-on-read):** `ArtifactTree` gains a private `keyMaps: Map<NodeId, Map<string, NodeId>>` cache — NOT stored on nodes (keeps `ArbNode`/StoredArtifact format byte-identical). New method:

```ts
  /** O(1) child lookup: arrays by index, objects via a lazily built key map. */
  childByKey(parentId: NodeId, key: string): ArbNode | undefined {
    const parent = this.nodes.get(parentId);
    if (!parent) return undefined;
    if (parent.kind === "array") {
      const i = Number(key);
      if (!Number.isInteger(i) || i < 0 || i >= parent.childIds.length) return undefined;
      return this.nodes.get(parent.childIds[i]);
    }
    let map = this.keyMaps.get(parentId);
    if (!map) {
      map = new Map();
      for (const cid of parent.childIds) map.set(String(this.nodes.get(cid)!.key), cid);
      this.keyMaps.set(parentId, map);
    }
    const cid = map.get(key);
    return cid === undefined ? undefined : this.nodes.get(cid);
  }
```

Invalidation — `this.keyMaps.delete(<parentId>)` at EVERY point a node's children change: `insertChild` (parent), `removeChild` (parent), `moveNode` (BOTH old and new parent), `replaceValue` (the node itself), and `deleteDescendants` deletes each removed id's own entry; `restore(snap)` and `fromStored` do `keyMaps.clear()` (fromStored: initialize empty). Arrays never populate the map (index path), so `renumberArray` needs nothing.

## Task 6: AG-UI adapter (STATE_SNAPSHOT / STATE_DELTA over the event log)

**Files:** create `src/ag-ui.ts`; modify `src/index.ts` (barrel export); create `test/m17-ag-ui.test.ts`

**Why:** AG-UI standardized "shared JSON state synchronized via STATE_SNAPSHOT + RFC 6902 JSON-Patch STATE_DELTA events" and is adopted by Microsoft Agent Framework, Google ADK, Mastra, PydanticAI. Arbor's events already record JSON-Pointer paths + before/after — the mapping is nearly mechanical, and this adapter is the cheapest interop/visibility move for a published arborkit. Zero new dependencies: we emit plain objects shaped like AG-UI events, we do NOT depend on any AG-UI SDK.

**Mapping locked:** `set` → `{op:"replace", path, value: after}` · `insert` → `{op:"add", path, value: after}` (array adds at index — matches) · `remove` → `{op:"remove", path}` · `move` → `{op:"move", from: fromPath, path: toPath}`. Events without recorded paths (pre-M7) map to `null` and are skipped with a documented note.

```ts
import type { Json } from "./types";
import type { ArtifactTree } from "./artifact-tree";
import type { EventLog, MutationEvent } from "./event-log";

/** RFC 6902 operation (the subset Arbor emits). */
export type JsonPatchOp =
  | { op: "replace" | "add"; path: string; value: Json }
  | { op: "remove"; path: string }
  | { op: "move"; from: string; path: string };

export interface AgUiStateSnapshot { type: "STATE_SNAPSHOT"; snapshot: Json; }
export interface AgUiStateDelta { type: "STATE_DELTA"; delta: JsonPatchOp[]; }

/** One mutation event as an RFC 6902 op, or null for pre-M7 events without paths. */
export function toJsonPatch(e: MutationEvent): JsonPatchOp | null {
  switch (e.kind) {
    case "set":
      return e.path === undefined ? null : { op: "replace", path: e.path, value: e.after ?? null };
    case "insert":
      return e.path === undefined ? null : { op: "add", path: e.path, value: e.after ?? null };
    case "remove":
      return e.path === undefined ? null : { op: "remove", path: e.path };
    case "move":
      return e.fromPath === undefined || e.toPath === undefined
        ? null
        : { op: "move", from: e.fromPath, path: e.toPath };
  }
}

/** The full current state as an AG-UI STATE_SNAPSHOT event. */
export function snapshotEvent(tree: ArtifactTree): AgUiStateSnapshot {
  return { type: "STATE_SNAPSHOT", snapshot: tree.toJson() };
}

/** Retained events with seq >= sinceSeq as ONE AG-UI STATE_DELTA event (pathless
 *  pre-M7 events are skipped). Returns the delta plus the next since-cursor. */
export function deltaSince(log: EventLog, sinceSeq: number): { event: AgUiStateDelta; nextSeq: number } {
  const delta: JsonPatchOp[] = [];
  for (const e of log.since(sinceSeq)) {
    const op = toJsonPatch(e);
    if (op) delta.push(op);
  }
  return { event: { type: "STATE_DELTA", delta }, nextSeq: log.length() };
}
```
